# W.I.P

This is where i will release all my unreleased tools ranging from july-november, by testing if it works, then publishing it.

## PFDBackport

Converts Gens CAB Compressed PFD's to Unleashed.

**Usage:** Drag and drop PFD file, then specify output path.

---

## col2fbx

Exports collision files (.phy.hkx) to FBX format.

**Usage:** Drag and drop .phy.hkx file onto col2fbx.exe

- Supports Unleashed (big endian)
- Removes `.phy` extension from output filename

---

## fbx2col

Converts FBX files to collision format (.phy.hkx).

**Usage:** Drag and drop .fbx file onto fbx2col.exe

- Add `_rigid` to filename for rigid body collision
